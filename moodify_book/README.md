## 核心内容速查

### 情绪潮汐四阶段

| 阶段 | 核心感受 | 音乐特征 |
|------|----------|----------|
| **蜷缩** | 紧、沉、冷 | 低频包裹、循环不变 |
| **迷茫** | 不确定、悬浮 | 中频移动、频率不稳 |
| **觉醒** | 看见了什么 | 高频出现、转折点 |
| **舒展** | 打开、呼吸 | 全频展开、空间打开 |

### 品牌Slogan

```
主Slogan：
Stay in the flow, stay in the soul.
不为抵达，只为停留。

副Slogan：
情绪的潮汐，终将抵达彼岸。

品牌声明：
Moodify 不生产"好听的背景音"。
我们制造"情绪的镜像"。
```

### 音乐边界四象限

```
象限一：听觉基调
  新极简主义 + 奶油质感
  → 磨砂玻璃感、Ambient、Lo-Fi

象限二：旋律逻辑
  呼吸感 + 留白
  → 长音、慢速、不洗脑

象限三：编曲结构
  演化循环结构
  → 蜷缩loop → 转折点 → 舒展loop

象限四：情绪调性
  中性偏冷 + 底温
  → 冬天的热水
```
