## 6.1 品牌的三个声音原则

Moodify 的品牌语言，基于三个原则：

```
原则一：克制
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
我们不说"治愈"、"改善"、"解决"、"帮助你"。

我们说"陪伴"、"停留"、"呼吸"。

克制 = 不夸大
克制 = 尊重用户的状态

我们不是来"拯救"谁的。
我们只是来陪一会儿。


原则二：诚实
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
我们承认我们的音乐不会"让一切变好"。

我们不会说"闭上眼睛，一切都会过去"。

我们会说"这个时刻很难。我们在这里。"

诚实 = 不虚假承诺
诚实 = 承认局限


原则三：邀请
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
我们不说"你应该"、"你需要"、"你必须"。

我们说"你可以"、"你想吗"、"你想在哪里"。

邀请 = 不强迫
邀请 = 给予选择权

用户可以选择蜷缩、迷茫、觉醒、舒展。
用户可以选择停留、离开、回来。
用户永远有选择。
```

## 6.2 品牌宣言（Slogan）

### 主Slogan

```
Stay in the flow, stay in the soul.

不为抵达，只为停留。
```

**解读：**

```
Stay in the flow
= 待在潮汐里
= 不要试图逃离情绪的潮汐
= 让它来，让它走

Stay in the soul
= 待在灵魂里
= 不要跑到外面的世界去找答案
= 答案在你这里

Stay in the flow, stay in the soul
= 在潮汐里，在灵魂里
= 这两件事是同一件事

不为抵达，只为停留
= 潮汐不是要去哪里
= 潮汐只是潮汐
= 停留本身，就是意义
```

---

### 副Slogan

```
情绪的潮汐，终将抵达彼岸。
```

**解读：**

```
情绪的潮汐
= 情绪是潮汐，不是直线
= 会来，会走

终将抵达彼岸
= 不是"你必须抵达"
= 而是"潮汐的方向，是舒展的方向"
= 这是自然规律，不是我们的承诺
= 蜷缩之后，一定是舒展
= 潮汐一定会退下去
```

---

### 品牌声明

```
Moodify 不生产"好听的背景音"。

我们制造"情绪的镜像"。

我们认可每一种负面情绪的合理性，
并致力于通过频率与旋律，
将"蜷缩"的时刻转化为"舒展"的契机。

但"转化"不是我们的目的。

"陪伴"才是。
```

## 6.3 文案语调（Voice & Tone）

### 语调地图

```
Moodify 的语调，在两个轴上：

Y轴（温度）：
冷 ←━━━━━━━━━━━→ 暖
（不热情、不压迫）     （温和、接纳）

X轴（姿态）：
低 ←━━━━━━━━━━━━━→ 高
（不说教、不教训）     （教导、建议）

Moodify 的位置：
              暖
               ↑
               |
               |
冷 ←←←←←←←←←←←|←←←←←←←← 高
               |     (X轴)
               |
               ↓
            不确定
              (X轴)

实际位置：
         暖 + 低姿态 + 不确定

这就是"Moodify的声音"：
温和、克制、邀请、不确定（承认我们不知道）
```

### 具体语调示例

#### 场景一：用户首次打开App

```
✗ 过度热情：
"欢迎来到Moodify！这里有1000+首治愈音乐，让你的心灵得到安慰！"

✗ 过于功能：
"请选择你的情绪状态，我们将为你推荐最佳音乐。"

✓ Moodify语调：
"你今天想待在哪里？"
```

#### 场景二：用户选择"蜷缩"

```
✗ 过度引导：
"别担心，我们来帮你放松。深呼吸，开始你的治愈之旅。"

✗ 过于疏离：
"你选择了蜷缩。音乐已开始播放。"

✓ Moodify语调：
"蜷缩是没问题的。
我在这里。
音乐开始了。"
```

#### 场景三：用户听了一会儿，选择退出

```
✗ 错误挽留：
"你确定要离开吗？你的情绪还没调整好呢！"

✓ Moodify语调：
"好的。
随时回来。
我在这里。"
```

"治愈"——我们不做治愈，我们做陪���
"改善"——我们不做改善，我们做接纳
"解决"——我们不解决问题，我们容纳问题
"治疗"——我们不是医疗，我们不是治疗
"应该"——我们不说"你应该开心"
"必须"——我们不说"你必须放松"
"正确"——我们不说"正确的生活方式"
"更好"——我们不说"成为更好的自己"
"加油"——我们不说"加油，一切都会好的"

✓ 我们说：

## 6.5 品牌视觉语言

### 色彩系统

```
Moodify 的色彩体系：低饱和度的"情绪色谱"

主色：灰蓝
━━━━━━━━━━━━━━━━━
情绪对应：蜷缩 / 迷茫
色值：#8B9EB7（灰蓝）
使用场景：蜷缩专辑封面、App主色

辅色：淡灰
━━━━━━━━━━━━━━━━━
情绪对应：迷茫 / 觉醒
色值：#AEB8C4（淡灰）
使用场景：迷茫专辑封面

强调色：透光蓝
━━━━━━━━━━━━━━━━━
情绪对应：觉醒 / 舒展
色值：#B8C5D4（透光蓝）
使用场景：觉醒/舒展专辑封面

渐变色：从冷灰蓝 → 透光蓝
━━━━━━━━━━━━━━━━━
代表情绪从"蜷缩"到"舒展"的过渡
```

### 色彩的情绪映射

```
蜷缩：深灰蓝 #6B7A8F
     紧、沉、冷

迷茫：灰蓝 #8B9EB7
     不确定、悬浮

觉醒：淡灰蓝 #A8B8C9
     看见了什么

舒展：透光蓝 #C4D4E4
     打开、松开
```

### 视觉意象

```
Moodify 的视觉，是"模糊的清晰"：

✓ 我们用：
——雾里的灯
——雨天的窗
——深夜的灯塔
——海边的灯
——模糊的远景
——只有轮廓，没有细节

✗ 我们不用：
——阳光
——笑脸
——彩虹
——向上指的箭头
——任何"积极向上"的意象
```

## 6.7 本章小结

```
品牌声音三原则：
1. 克制 - 不夸大，不拯救
2. 诚实 - 不虚假承诺，承认局限
3. 邀请 - 不强迫，给予选择权

品牌Slogan：
- 主：Stay in the flow, stay in the soul.
- 副：不为抵达，只为停留。

语调坐标：
- 温度：冷 → 暖（温和）
- 姿态：低 → 高（不说教）
- 实际位置：温和 + 不说教 + 不确定

禁止语：
- 不说"治愈"、"改善"、"解决"
- 说"陪伴"、"停留"、"呼吸"

视觉语言：
- 低饱和度色彩
- 灰蓝体系
- 模糊的清晰意象
- 不使用阳光、笑脸等"积极意象"
```
