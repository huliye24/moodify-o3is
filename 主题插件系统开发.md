## 一、项目概述

### 1.1 目标
将 Moodify 的主题系统从硬编码方式重构为**可扩展的插件系统**，允许第三方开发者创建、发布和分享主题包。

### 1.2 核心特性

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 主题插件注册 | P0 | 第三方主题可注册到系统 |
| 主题切换 | P0 | 支持在已安装主题间自由切换 |
| 主题预览 | P1 | 切换前预览主题效果 |
| 主题市场 | P1 | 在线浏览和安装主题 |
| 用户主题 | P1 | 用户可创建自定义主题 |
| 导入/导出 | P2 | 分享和备份主题配置 |

### 1.3 技术选型

```

├── React 18.x          # 前端框架
```

### 2.2 核心模块设计

#### 2.2.1 主题配置接口 (ThemeSchema)

```typescript
// packages/core/src/types/theme.ts

export interface ThemeColors {
  // 页面背景
  bgBase: string;
  bgSurface: string;
  bgElevated: string;
  
  // 情绪卡片颜色
  coil: { color: string; gradient: string };
  lost: { color: string; gradient: string };
  awaken: { color: string; gradient: string };
  expand: { color: string; gradient: string };
  
  // 文字颜色
  textPrimary: string;
  textSecondary: string;
  textMuted: string;
  
  // 强调色
  accent: string;
  
  // 边框
  border: string;
  borderHover: string;
}

export interface ThemeEffects {
  particles?: {
    enabled: boolean;
    type: 'sakura' | 'snow' | 'leaves' | 'none';
    count?: number;
    colors?: string[];
  };
  gradients?: {
    enabled: boolean;
    intensity?: number;
  };
}

export interface ThemeManifest {
  id: string;                    // 主题唯一标识 'moodify-theme-sakura'
  name: string;                  // 显示名称 '樱花'
  description?: string;          // 主题描述
  author?: string;               // 作者信息
  version: string;               // 版本号 '1.0.0'
  thumbnail?: string;             // 预览图URL
  colors: ThemeColors;            // 颜色配置
  effects?: ThemeEffects;        // 特效配置
  builtIn?: boolean;             // 是否内置主题
  cssVariables?: Record<string, string>; // 自定义CSS变量
}
```

#### 2.2.2 主题注册表 (ThemeRegistry)

```typescript
// packages/core/src/registry/ThemeRegistry.ts

class ThemeRegistry {
  private themes: Map<string, ThemeManifest> = new Map();
  private listeners: Set<() => void> = new Set();
  
  // 注册主题
  register(theme: ThemeManifest): void;
  
  // 注销主题
  unregister(themeId: string): void;
  
  // 获取主题
  get(themeId: string): ThemeManifest | undefined;
  
  // 获取所有主题
  getAll(): ThemeManifest[];
  
  // 订阅变化
  subscribe(listener: () => void): () => void;
  
  // 批量注册
  registerBatch(themes: ThemeManifest[]): void;
}

// 导出单例
export const themeRegistry = new ThemeRegistry();
```

#### 2.2.3 主题引擎 (ThemeEngine)

```typescript
// packages/core/src/engine/ThemeEngine.ts

class ThemeEngine {
  private currentThemeId: string | null = null;
  
  // 应用主题到DOM
  apply(theme: ThemeManifest): void;
  
  // 预览主题（临时应用，不保存）
  preview(theme: ThemeManifest): void;
  
  // 取消预览，恢复当前主题
  cancelPreview(): void;
  
  // 获取当前主题ID
  getCurrentThemeId(): string | null;
  
  // 重置为默认主题
  reset(): void;
}
```
