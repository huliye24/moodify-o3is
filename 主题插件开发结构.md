> 文档版本：v1.0

---

## 一、概述

### 1.1 什么是主题插件？

主题插件是一个 **npm 包**，它定义了 Moodify 应用的外观样式，包括：
- 颜色配置（背景、文字、强调色等）
- 粒子特效（樱花、雪花、落叶等）
- 自定义 CSS 变量

### 1.2 快速开始

```bash
# 创建主题插件项目
npx create-moodify-theme my-theme

# 进入目录
cd my-theme

# 开发调试
npm run dev

# 构建发布
npm run build
npm publish
```

## 三、核心文件详解

### 3.1 入口文件 `src/index.ts`

这是插件的主入口，负责向 Moodify 主应用注册主题。

```typescript
/**
 * moodify-theme-xxx/src/index.ts
 * 
 * 主题插件入口文件
 * 此文件必须导出 default，且返回 defineTheme() 的结果
 */

import { defineTheme } from '@moodify/core';
import { theme } from './theme';

export default defineTheme(theme);
```

### 3.2 主题配置 `src/theme.ts`

定义主题的所有视觉属性。

```typescript
/**
 * moodify-theme-xxx/src/theme.ts
 * 
 * 主题配置
 */

import { ThemeManifest, ThemeColors, ThemeEffects } from '@moodify/core';

// 颜色配置
const colors: ThemeColors = {
  // ============================================
  // 背景色 - 页面层级的背景颜色
  // ============================================
  bgBase: '#f0ede8',       // 主背景
  bgSurface: '#e5dfd8',    // 卡片/面板背景
  bgElevated: '#f7f4f0',  // 浮层/弹出框背景

  // ============================================
  // 情绪卡片颜色 - 四种情绪对应的颜色
  // ============================================
  coil: { 
    color: '#C47B8E', 
    gradient: 'linear-gradient(145deg, #a86050 0%, #985040 50%, #C47B8E 100%)' 
  },
  lost: { 
    color: '#D4A574', 
    gradient: 'linear-gradient(145deg, #b88560 0%, #a87550 50%, #D4A574 100%)' 
  },
  awaken: { 
    color: '#E8B86C', 
    gradient: 'linear-gradient(145deg, #c89858 0%, #b88848 50%, #E8B86C 100%)' 
  },
  expand: { 
    color: '#D8C8B8', 
    gradient: 'linear-gradient(145deg, #c8b8a8 0%, #b8a898 50%, #D8C8B8 100%)' 
  },

  // ============================================
  // 文字颜色 - 不同优先级的文字颜色
  // ============================================
  textPrimary: '#3d2d2d',   // 主要文字（深色）
  textSecondary: '#5d4d4d', // 次要文字
  textMuted: '#8d7d7d',    // 弱化文字（注释、提示）

  // ============================================
  // 强调色 - 用于按钮、链接、焦点等
  // ============================================
  accent: '#a86850',

  // ============================================
  // 边框 - 卡片、按钮的边框颜色
  // ============================================
  border: 'rgba(61, 45, 45, 0.1)',        // 正常状态
  borderHover: 'rgba(61, 45, 45, 0.2)',  // 悬停状态
};

// 特效配置
const effects: ThemeEffects = {
  particles: {
    enabled: true,
    type: 'sakura',           // 'sakura' | 'snow' | 'leaves' | 'none'
    count: 40,                // 粒子数量（建议 20-80）
    colors: [                 // 粒子颜色（建议 4-8 种）
      '#FFB7C5',
      '#FFC8D5',
      '#FFD8E8',
      '#FFE8F0',
    ],
  },
  gradients: {
    enabled: true,
    intensity: 0.6,           // 渐变强度 0-1
  },
};

// 主题清单
export const theme: ThemeManifest = {
  id: 'moodify-theme-xxx',     // 唯一标识（格式：moodify-theme-{name}）
  name: '我的主题',              // 显示名称
  description: '这是一个精美的自定义主题，特点是...', // 描述
  author: '开发者名称',         // 作者
  version: '1.0.0',            // 版本号（遵循 semver）
  thumbnail: './assets/thumbnail.png', // 预览图路径
  
  colors,
  effects,
  
  // 可选：自定义 CSS 变量
  cssVariables: {
    '--custom-radius': '12px',
    '--custom-shadow': '0 4px 20px rgba(0,0,0,0.1)',
  },
};
```

## 四、类型定义

### 4.1 ThemeManifest（主题清单）

```typescript
interface ThemeManifest {
  /** 主题唯一标识，格式：moodify-theme-{name} */
  id: string;
  
  /** 显示名称 */
  name: string;
  
  /** 主题描述 */
  description?: string;
  
  /** 作者信息 */
  author?: string;
  
  /** 版本号（遵循 semver） */
  version: string;
  
  /** 预览图路径 */
  thumbnail?: string;
  
  /** 颜色配置 */
  colors: ThemeColors;
  
  /** 特效配置 */
  effects?: ThemeEffects;
  
  /** 是否内置主题 */
  builtIn?: boolean;
  
  /** 自定义 CSS 变量 */
  cssVariables?: Record<string, string>;
}
```

### 4.2 ThemeColors（颜色配置）

```typescript
interface ThemeColors {
  // 背景色
  bgBase: string;        // 主背景
  bgSurface: string;      // 卡片/面板背景
  bgElevated: string;    // 浮层背景
  
  // 情绪卡片颜色
  coil: { color: string; gradient: string };
  lost: { color: string; gradient: string };
  awaken: { color: string; gradient: string };
  expand: { color: string; gradient: string };
  
  // 文字颜色
  textPrimary: string;    // 主要文字
  textSecondary: string;  // 次要文字
  textMuted: string;      // 弱化文字
  
  // 强调色
  accent: string;
  
  // 边框
  border: string;
  borderHover: string;
}
```

### 4.3 ThemeEffects（特效配置）

```typescript
interface ThemeEffects {
  particles?: {
    /** 是否启用粒子效果 */
    enabled: boolean;
    
    /** 粒子类型 */
    type: 'sakura' | 'snow' | 'leaves' | 'none';
    
    /** 粒子数量（建议 20-80） */
    count?: number;
    
    /** 粒子颜色数组 */
    colors?: string[];
  };
  
  gradients?: {
    /** 是否启用渐变背景 */
    enabled: boolean;
    
    /** 渐变强度 0-1 */
    intensity?: number;
  };
}
```

## 六、开发规范

### 6.1 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 包名 | `moodify-theme-{name}` | `moodify-theme-sakura` |
| 主题ID | 同包名 | `moodify-theme-sakura` |
| 显示名 | 中文或英文 | `樱花`、`Midnight Blue` |

### 6.2 颜色规范

- 使用 HEX 格式（如 `#FFB7C5`）
- 确保对比度满足 WCAG AA 标准
- 文字颜色应与背景有明显区分

### 6.3 版本规范

遵循 [Semantic Versioning](https://semver.org/lang/zh-CN/)：
- `MAJOR.MINOR.PATCH`
- 例如：`1.0.0` → `1.1.0` → `2.0.0`

### 6.4 发布规范

1. 测试主题在所有页面正常工作
2. 添加清晰的 README 文档
3. 准备一张高质量的预览图
4. 选择合适的许可证
5. 发布到 npm：`npm publish`

## 七、工具函数

### 7.1 颜色工具 `src/utils/colorUtils.ts`

```typescript
/**
 * 颜色工具函数
 */

// HEX 转 RGB
export function hexToRgb(hex: string): { r: number; g: number; b: number } {
  const result = /^#?([a-f\d]{2})([a-f\d]{2})([a-f\d]{2})$/i.exec(hex);
  return result
    ? {
        r: parseInt(result[1], 16),
        g: parseInt(result[2], 16),
        b: parseInt(result[3], 16),
      }
    : { r: 0, g: 0, b: 0 };
}

// RGB 转 HEX
export function rgbToHex(r: number, g: number, b: number): string {
  return '#' + [r, g, b].map(x => x.toString(16).padStart(2, '0')).join('');
}

// 调整颜色亮度
export function adjustBrightness(hex: string, percent: number): string {
  const { r, g, b } = hexToRgb(hex);
  const adjust = (c: number) => Math.min(255, Math.max(0, c + (255 * percent) / 100));
  return rgbToHex(adjust(r), adjust(g), adjust(b));
}

// 生成渐变色
export function generateGradient(color1: string, color2: string, angle: number = 145): string {
  return `linear-gradient(${angle}deg, ${color1} 0%, ${color2} 100%)`;
}
```

### 7.2 渐变工具 `src/utils/gradients.ts`

```typescript
/**
 * 渐变工具函数
 */

export function generateMoodGradient(
  baseColor: string,
  darkerColor: string,
  accentColor: string
): {
  coil: string;
  lost: string;
  awaken: string;
  expand: string;
} {
  return {
    coil: `linear-gradient(145deg, ${baseColor} 0%, ${darkerColor} 50%, ${accentColor} 100%)`,
    lost: `linear-gradient(145deg, ${adjustBrightness(baseColor, 10)} 0%, ${adjustBrightness(darkerColor, 5)} 50%, ${baseColor} 100%)`,
    awaken: `linear-gradient(145deg, ${adjustBrightness(baseColor, 20)} 0%, ${adjustBrightness(accentColor, -10)} 50%, ${adjustBrightness(baseColor, 10)} 100%)`,
    expand: `linear-gradient(145deg, ${adjustBrightness(baseColor, 30)} 0%, ${adjustBrightness(accentColor, 10)} 50%, ${adjustBrightness(baseColor, 20)} 100%)`,
  };
}
```

## 九、常见问题

### Q1: 如何调试主题？

```bash
# 1. 克隆开发模板
git clone https://github.com/moodify/theme-template.git my-theme

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 在浏览器中预览（自动打开）
```

### Q2: 如何分享我的主题？

1. 发布到 npm
2. 在 GitHub 上创建仓库
3. 在 Moodify 社区提交主题

### Q3: 如何更新主题？

```bash
# 1. 修改代码
# 2. 更新版本号
npm version patch  # 1.0.0 -> 1.0.1

# 3. 发布
npm publish
```
