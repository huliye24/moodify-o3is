# 工作日志

## 说明

此文件夹包含开发过程中的工作日志。每次完成任务后，应在此创建或更新相应的日志文件。

## 日志命名规范

```
YYYY-MM-DD-任务名称.md
```

## 日志格式

```markdown
# YYYY-MM-DD - 任务名称

## 完成内容

- [x] 完成的任务1
- [x] 完成的任务2

## 遇到的问题

- 问题1：描述及解决方案
- 问题2：描述

## 下一步

- [ ] 下一个任务1
- [ ] 下一个任务2

## 耗时

- 预计: X 小时
- 实际: X 小时

---

## 更新记录

| 日期 | 内容 | 负责人 |
|------|------|--------|
| YYYY-MM-DD | 创建日志 | Name |
```
