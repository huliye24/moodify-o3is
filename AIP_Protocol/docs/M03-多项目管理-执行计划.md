## 📋 Week 1: 项目注册表（6/14 - 6/20）

### 目标
创建项目注册表，让 AIP 协议能同时管理多个项目。

### 为什么需要多项目管理？

**现状**：
- 一个 `AIP_Protocol/` 只能管一个项目
- Moodify 有多个子项目（backend、frontend、electron）
- 每个项目单独管理，效率低

**目标**：
```
AIP 协议仓库（中央）
├── projects/
│   ├── moodify/          # 主项目
│   │   ├── AIP_Protocol/ # 项目特定协议（可覆盖）
│   │   └── config.yaml   # 项目配置
│   ├── backend/          # 后端服务
│   │   └── config.yaml
│   └── frontend/         # 前端应用
│       └── config.yaml
├── shared/               # 共享协议（所有项目继承）
│   ├── 0_决策/
│   ├── 1_roadmap/
│   ├── 2_任务/
│   └── 3_共享/
└── projects-registry.yaml  # 项目注册表
```

```yaml
      language: "TypeScript + Go"
      start_date: "2026-01-01"
      status: "active"
      tags: ["fullstack", "electron", "ai"]

  - id: "backend"
    name: "Moodify Backend"
    path: "./backend/"
    protocol_version: "1.0"
    inherit_from: "shared/base"
    metadata:
      language: "Go"
      status: "active"

  - id: "frontend"
    name: "Moodify Frontend"
    path: "./frontend/"
    protocol_version: "1.0"
    inherit_from: "shared/base"
    metadata:
      language: "TypeScript"
      status: "active"
```

**解析器实现**：
```javascript
// tools/load-projects.js
function loadProjects() {
  const registry = readYAML('projects-registry.yaml');
  const projects = [];

  for (const def of registry.projects) {
    const project = {
      ...def,
      // 合并共享协议
      ...mergeSharedProtocol(def),
      // 应用覆盖
      ...applyOverrides(def),
      // 计算完整路径
      fullPath: path.resolve(def.path)
    };
    projects.push(project);
  }

  return projects;
}
```


**实现**：
```javascript
function mergeProtocol(project) {
  const merged = {
    decisions: [...shared.decisions, ...project.overrides.decisions],
    roadmap: deepMerge(shared.roadmap, project.overrides.roadmap),
    tasks: [...shared.tasks, ...project.tasks],
    shared: { ...shared.shared }
  };
  return merged;
}
```

## 📋 Week 2: 协议共享机制（6/21 - 6/27）

### 目标
让多个项目能共享同一套协议配置，减少重复。

### 核心问题

**没有共享的问题**：
```
Project A: 复制了一套协议 → 修改后无法同步
Project B: 又复制了一套 → 很快不一致
Project C: 自己改了一版 → 与 A、B 都不兼容
```

**解决方案**：
```
Shared Protocol (中央)
├── 0_决策/          ← 公共决策
├── 1_roadmap/       ← 公共路线
├── 2_任务/          ← 公共任务模板
└── 3_共享/          ← 公共钩子

Project A
├── 继承 Shared
├── 添加 A-specific 决策
└── 覆盖 A-specific 路线

Project B
├── 继承 Shared
└── 完全使用 Shared（无覆盖）
```

**协议加载器**：
```javascript
class ProtocolLoader {
  constructor(projectId) {
    this.project = loadProject(projectId);
    this.shared = loadProtocol(this.project.inherit_from);
    this.projectSpecific = loadProtocol(projectId);
  }

  getDecisions() {
    // 合并：共享 + 项目特定（后者优先）
    return [...this.shared.decisions, ...this.projectSpecific.decisions];
  }

  getTasks() {
    // 合并，去重
    return dedupe([...this.shared.tasks, ...this.projectSpecific.tasks]);
  }

  getHooks() {
    // 合并，项目特定可添加额外钩子
    return [...this.shared.hooks, ...this.projectSpecific.hooks];
  }
}
```

**聚合器**：

```javascript
// tools/aggregate-projects.js
function aggregateAllProjects() {
  const projects = loadAllProjects();
  const stats = {
    totalTasks: 0,
    completedTasks: 0,
    totalDecisions: 0,
    averageHealth: 0,
    byProject: {}
  };

  for (const project of projects) {
    const projectStats = getProjectStats(project);
    stats.byProject[project.id] = projectStats;

    stats.totalTasks += projectStats.totalTasks;
    stats.completedTasks += projectStats.completedTasks;
    stats.totalDecisions += projectStats.decisions.length;
  }

  stats.completionRate = stats.completedTasks / stats.totalTasks;
  stats.averageHealth = average(projects.map(p => p.healthScore));

  return stats;
}
```

```javascript
// tools/migrate-protocol.js
const migrations = {
  '1.0.0 -> 1.1.0': (project) => {
    // 1. 添加决策影响分析字段
    // 2. 更新任务 Schema
  },
  '1.1.0 -> 2.0.0': (project) => {
    // 1. 创建 projects-registry.yaml
    // 2. 重构目录结构
    // 3. 迁移任务到新格式
  }
};

function migrate(projectPath, fromVersion, toVersion) {
  const key = `${fromVersion} -> ${toVersion}`;
  const migration = migrations[key];

  if (!migration) {
    throw new Error(`无此迁移: ${key}`);
  }

  // 执行迁移
  migration(projectPath);

  // 更新 VERSION
  writeVersion(projectPath, toVersion);
}
```

**检查器**：
```javascript
function checkCompatibility(projectVersion, latestVersion) {
  const comparison = compareVersions(projectVersion, latestVersion);

  if (comparison.major < 0) {
    return {
      compatible: false,
      reason: `破坏性变更：需升级到 ${latestVersion}`,
      migration: `运行: node tools/migrate-protocol.js`
    };
  }

  if (comparison.minor < 0) {
    return {
      compatible: true,
      warning: `建议升级到 ${latestVersion} 以使用新功能`,
      newFeatures: getNewFeatures(projectVersion, latestVersion)
    };
  }

  return { compatible: true };
}
```
