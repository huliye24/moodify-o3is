# AIP 协议项目目录结构

```
AIP_Protocol/
├── 0_决策/                    # 不可变意图（宪法级文件）
│   ├── 愿景.yaml              # 项目终极目标
│   ├── 约束.yaml              # 硬性限制条件
│   └── 价值观.yaml            # 核心价值判断
│
├── 1_roadmap/                 # 可演化的科技树
│   ├── 技术栈.yaml            # 技术选型
│   ├── 架构图.mmd             # Mermaid 架构图
│   ├── 组件定义.yaml          # 组件依赖关系
│   ├── 演进路线/              # 版本演进规划
│   │   ├── M01-核心框架.md
│   │   ├── M02-创作流程.md
│   │   └── ...
│   └── 优化记录/              # 架构优化日志
│       └── YYYY-MM-DD-优化描述.md
│
├── 2_任务/                    # 原子任务系统
│   ├── T20260413_001_初始化AIP协议目录.json
│   ├── T20260413_002_创建决策文档.json
│   └── ...
│
├── 3_共享/                    # 全局状态与钩子
│   ├── HOOKS.yaml             # 任务完成后必须执行的自检清单
│   ├── 事件日志.yaml          # 所有任务执行记录
│   ├── 仪表盘.md              # 全局进度可视化
│   └── 状态快照/              # 定期状态备份
│
├── contracts/                 # 契约文件（约束条件）
│   ├── global-constraints.yaml
│   ├── performance.yaml
│   └── api-contracts.yaml
│
├── docs/                      # 协议文档
│   ├── AIP_OVERVIEW.md        # 协议总览
│   ├── NODE_DESIGN.md         # Node 设计规范
│   └── AI_PROMPT_GUIDE.md     # AI 协作提示词模板
│
├── tools/                     # 自动化工具
│   ├── update-dashboard.js    # 更新仪表盘
│   ├── validate-tasks.js      # 验证任务格式
│   ├── update-decision-index.js
│   └── weekly-report.js
│
├── examples/                  # 示例项目
│   └── moodify/               # Moodify 作为 AIP 协议的示例项目
│       ├── 决策/              # 软链接到 ../decisions/
│       ├── 路线图/            # 软链接到 ../roadmap/
│       └── 任务/              # 软链接到 ../tasks/
│
└── AIP_CONSTITUTION.md        # 本文件，给 AI 的指令
