### 构建目标

| 平台 | 格式 | 打包工具 |
|------|------|----------|
| Windows | `.exe` (NSIS 安装包) | electron-builder |
| macOS | `.dmg` | electron-builder |
| Linux | `.AppImage` | electron-builder |

### 版本策略

- **语义版本**: `MAJOR.MINOR.PATCH`
- **发布节奏**: 每月一个小版本，每季度一个大版本
- **测试版本**: 每周发布测试版
- **自动更新**: 使用 electron-updater

### 约束契约

```yaml
# contracts/deployment.yaml
deployment:
  mode: "local_first"

  platforms:
    - windows_x64
    - macos_x64
    - macos_arm64
    - linux_x64

  build:
    output: "./release"
    format: "dir"  # 先输出目录，再打包

  versioning:
    scheme: "semver"
    auto_update: true

  constraints:
    max_bundle_size_mb: 200
    max_install_size_mb: 500
    min_free_space_mb: 1000
```

## 理由

### 本地优先的理由
- 保护用户隐私
- 无需网络即可使用
- 减少服务器成本
- 离线可用

### 可选云端的理由
- 提供数据备份能力
- 跨设备同步（付费功能）
- 未来可扩展 SaaS

## 影响范围

- 构建流程必须支持多平台
- 数据格式必须支持导入/导出
- 必须考虑离线场景
