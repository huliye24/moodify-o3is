# Moodify Player

情绪的潮汐，终将抵达彼岸。

## 简介

Moodify 是一款本地音乐播放器，采用情绪潮汐的设计理念。它不追求华丽的视觉效果，而是专注于提供安静、沉浸的音乐体验。

## 安装

1. 确保已安装 Python 3.7+

2. 安装依赖：

```bash
pip install -r requirements.txt
```

或者直接安装 PyQt5：

```bash
pip install PyQt5
```

## 运行

在 PyCharm 中打开项目，右键点击 `moodify_player.py`，选择 `Run 'moodify_player'`。

或者在终端中运行：

```bash
cd moodify_player
python moodify_player.py
```

## 使用方法

1. **添加音乐**：点击右上角"📁 音乐文件夹"按钮，选择包含音乐文件的文件夹
2. **播放音乐**：双击播放列表中的歌曲即可播放
3. **控制播放**：使用底部的播放/暂停、上一首、下一首按钮
4. **调节音量**：使用右下角的音量滑块

## 音乐文件夹

首次运行时，程序会在同目录下创建 `music` 文件夹。

你也可以点击"📁 音乐文件夹"按钮选择其他位置。

支持的格式：MP3、WAV、FLAC、OGG、M4A

## 界面说明

- **深色主题**：减少眼睛疲劳，适合夜间使用
- **极简设计**：专注于音乐本身，不打扰
- **情绪配色**：使用低饱和度的灰蓝色系，呼应情绪潮汐主题

## 常见问题

**Q: 为什么播放不了音乐？**
A: 请确保音乐文件夹中有支持格式的音乐文件（MP3、WAV、FLAC、OGG、M4A）。

**Q: 没有声音？**
A: 检查音量是否调高，以及系统音量设置。

**Q: 界面显示异常？**
A: 请确保使用支持高DPI的显示器，或在系统设置中调整缩放。
